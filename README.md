# saas_app

A subscription-based SaaS Flutter app with Supabase auth.

## Getting Started

### Running on Web

To avoid the "Falling back to CPU-only rendering" warning (when WebGL isn't available), use the HTML renderer:

```bash
flutter run -d chrome --web-renderer=html
```

Or use the "Flutter (Chrome)" launch config in VS Code/Cursor, which uses the HTML renderer by default.
